run.sh

A Makefile replacement written in pure bash. No DSL, no dependencies, just scripts.

curl -fsSL https://raw.githubusercontent.com/imgproxy/run-sh/main/install.sh | bash -e -g
  • No DSL to learn — it's bash, all the way down.
  • No runtime dependency, no build step, no framework.
  • run is generated and regenerable; your task logic lives in bin/, which re-installing never touches.
  • Project-wide helpers and variables go in .runrc.
  • Compose tasks with depends_on task1 task2 instead of Make's recipe graph.
  • Optional run global command finds your project from any subdirectory, like git finds .git.

A task looks like this

#!/usr/bin/env bash
# bin/greet.sh — sourced by ./run, not executed directly.

description() { echo "Greet someone"; }

help() {
  cat <<'EOF'
Usage: ./run greet <name>
EOF
}

main() {
  echo "Hello, ${1:?name required}!"
}

Usage

./run              # list available tasks
./run help <task>  # show a task's usage
./run <task> [args...]
./run new <task>   # scaffold a new task file
